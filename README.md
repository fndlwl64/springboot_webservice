# WEB_COMMUNITY

- 개인 프로젝트
- 간단한 커뮤니티 게시판
- aws ec2에 무중단 배포 중

# Tech

`JAVA SDK 11` `SpringBoot 2.7.1` `Mustache` `JPA` `H2` `MariaDB` `AWS EC2` `AWS RDS` `Travis CI` `Nginx`
